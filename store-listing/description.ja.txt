GitHub のコミットネットワークを、フォークも含めて
VSCode の Git Graph 風のビューアで一望できる Chrome 拡張機能です。

GitHub 標準のネットワークグラフは概観をつかむには良いのですが、
いざコミットの中身を読もうとすると別ページを何度も行き来する
必要があります。Git Graph for GitHub は、グラフとコミット詳細
パネルを 1 つのタブに並べて表示するので、リポジトリの履歴と
フォーク全体の関係を、タブを離れずに調べられます。

主な機能

・任意の公開 GitHub リポジトリのコミットグラフを専用タブで表示
・大きな履歴でも軽快に動く仮想スクロール
・フォーク対応:「Main only」(対象リポジトリのブランチのみ) と
 「Include forks」(ネットワーク全体) を切り替え可能
・コミット詳細パネル: 親 / 子コミットへの移動、ファイルごとの
 追加・削除行数とステータスを表示
・右クリックで SHA / コミットメッセージのコピー、親コミットへ
 のジャンプ
・ツールバーポップアップに「最近開いたリポジトリ」を保持
・ダーク / ライトテーマ
・Chrome の言語設定に応じて英語 / 日本語 UI を自動切り替え

こんな人におすすめ

・フォークが多い OSS のメンテナ / コントリビューターで、
 誰がどこで分岐したかを素早く把握したい方
・VSCode の Git Graph 拡張機能の使い勝手をブラウザでも
 そのまま味わいたい方
・コントリビュート前にプロジェクトの構造をざっと把握したい方

プライバシー

・解析・トラッキング・リモートコードの読み込みは一切なし
・通信はブラウザから github.com および api.github.com への直接
 アクセスのみで、第三者サーバーを経由することはありません
・ローカルに保存するのは設定 (テーマ、デフォルトのグラフ
 モード) と「最近開いたリポジトリ」のリストのみ
・閲覧中のページにコンテンツスクリプトを注入することは
 ありません

権限について

・storage: 設定と最近のリポジトリ一覧をローカルに保存
・activeTab: ポップアップを開いたときに現在のタブの URL を
 読み取り、そのリポジトリをグラフ表示できるようにするため
・github.com / api.github.com へのアクセス: コミットネット
 ワークおよびコミットごとの詳細を必要なときに取得 (未認証
 のため GitHub の 60 リクエスト/時の制限内で動作)
